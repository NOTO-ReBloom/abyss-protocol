# REV.23 改修内容

## 1. 世界設定の再統一

- 時代を西暦2096年に統一。
- 施設を深度8,021mの `PELAGIC CORE A-13` として再構成。
- 真舟湊、伊波澪、真舟遙の関係と、2093年の事故を導入サイトへ反映。
- 「深層合唱波は、異なる情報を持つ二人の合意にだけ反応する」という裏設定を、章構成・導入・任務ログへ接続。

## 2. 操作者画面の全面再設計

- 旧来の平面的なウェブカード表現を廃止。
- 耐圧制御卓、金属フレーム、埋め込みモニター、ボルト、ガラス反射、計器、配線盤を基調とした物理コンソールへ変更。
- ブラウザの拡大縮小に依存した描画を避け、日本語と数値を高精細に表示。
- 1280×720以上でページスクロールなしに操作可能。
- 9種類の装置画面を、物理スイッチ・ノブ・端子・レーダーとして再描画。

## 3. 成功・失敗フィードバック

- 装置解除時に圧力解放音、機械ロック音、発光、封鎖通知を追加。
- ミス時に警報、時間減算、危険表示、船殻・浸水系の音を追加。
- 章成功時に `SYSTEM STABILIZED` の全画面演出を追加。
- 失敗時に `SEAL FAILURE` の全画面演出を追加。
- 成功と失敗で音、色、残響、表示時間を明確に分離。

## 4. 音響

- Web Audio APIによる低周波、海流、機械振動、船殻の軋み、ソナー、警報、通信ノイズを再構成。
- 章、残り時間、ミス状態に応じて緊張音が変化。
- 主音量、環境音、操作・警報音を設定画面から調整可能。
- 音を停止してもゲーム進行と必須情報が失われない設計。

## 5. 解析員画面とPDF

- 解析員用HTML端末を、検索可能な緊急手順コンソールとして再設計。
- 操作者画面と解析員画面を相互に開ける導線を追加。
- 記号の読み方・名称をゲーム、HTML手順書、PDFから削除。
- 記号は均一な図形だけを表示し、呼び方を二人で作る余地を復元。
- PDFの章ルール、時間、装置数、REV表記を更新。

## 6. 導入映像と公式サイト

- 約56秒、8場面の導入映像を新規構成。
- 降下、事故、三年間の沈黙、死亡者の音声認証、二端末への情報分断、観測核の目的、封鎖開始を映像化。
- 低周波、警報、左右の音声接続、ノイズを場面別に生成。
- 無音再生、音量調整、スキップ、再視聴に対応。
- 主要背景画像をHTMLへ埋め込み、GitHub Pagesのパス問題を回避。
